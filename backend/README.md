# Backend

FastAPI scaffold for RoadSoS.

## Planned commands

```powershell
cd backend
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Ownership

Suyash owns the data, geo ranking, SQLite build, and core API implementation. Contract changes must be coordinated through `contracts/`.
