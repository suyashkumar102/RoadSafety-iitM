# Assumptions

- RoadSoS is built for the RoadSoS track only.
- The demo region is IIT Madras / Chennai.
- Production emergency contacts must be curated manually from reliable sources.
- AI may summarize user-provided incident details but cannot create emergency contacts.
- The first offline version will prioritize Chennai and a small portability sample.
- Exact judge point weights are not public, so `plan.md` uses an internal scoring proxy based on the stated evaluation criteria.
